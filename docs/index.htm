Commands

Who are we?

Bennett Adventure Team is a highly versatile multi-function Genshin-based bot for Discord which started development in 2022. It allows private support channels, which we call tickets, between your staff team and individuals who look for help. We also offer features such as music, suggestiosn, utilities, chat and fun commands. It’s highly customizable, giving you the freedom to be the perfect fit for your server. There is no need for complex setup procedures, or multiple commands to make simple actions. You can easily interact with the bot using Discord slash commands. With over 50 active servers, 28,000 happy users, and over 4,000 tickets to date, you can trust Bennett Adventure Team is going to be a great addition to your server.

General Utilities

Fun Commands

These are some commands made for entertainment:

  • /meme - Posts a random meme for you to laugh at
  • /slot - Rolls the slot machine, but of Genshin characters
  • /blackjack - Gambles with a virtual dealer
  • /8ball <question> - Consults 8ball to receive an answer
  • /f <reason> - Pays some respect by pressing F
  • /rate <thing> - Asks the bot to rate a thing
  • /howhot <user> - Returns the hotness percentage of a user
  • /hug <user> - Gives another user a warm hug

Useful Functions

Useful Functions

The following commands are used to setup and deal with tickets. To start using tickets instantly, use /ticket panel default and the bot will guide you through the procedures of setting up your ticket mailbox when you try to create your first ticket! Note that these commands are only available to members with Administrator permission, meaning only Admins can close/delete tickets.

Command Example Usage
ticket set category <category> /ticket set category category:Tickets Sets up the category that is used to hold all tickets.
ticket set log <channel> /ticket set log channel:ticket-log Sets up the channel that logs all the actions of tickets.
ticket panel default /ticket panel default Sends out the default ticket creation panel.
ticket panel custom <title> <description> <color> /ticket panel custom title:need help? description:click the button below to create a ticket color: #ff0000 Sends out a custom ticket creation panel.
close /close Closes the current ticket and prevents ticket author from viewing the ticket.
delete /delete Deletes the current ticket channel.
notify /notify Alerts the ticket user via DM to check the ticket.

The following commands are used to keep your server chat engaged and active by issuing random Genshin Impact topics to chat on! To setup the chat reviver ping, you may need Administrator permission.

Command Example Usage
topic /topic Get a topic related to Genshin Impact to chat on.
chatreviver enable <channel> <role> <minute> /chatreviver enable channel:general role:chat reviver ping minute:60 Sets up a revive message for a channel. Use the same command to change your settings.
chatreviver disable <channel> /chatreviver disable channel:general Removes a revive message channel.

[playlist|pl] <YouTube Playlist URL | Playlist Name> -playlist https://www.youtube.com/playlist?list=PL47U21_BOdXlmimq7jnYfjsTh6vJwLTkD Plays songs from a YouTube playlist

The following slash commands are for setting up a suggestion system for your server where memebers can offer suggestions to your server and your admins can manage their suggestions easily. All messages sent in a suggestion channel will be regarded as a suggestion and the bot will turn the message to an embed message with reactions and buttons. Note that members Administrator can only manage suggestions and the bot will ignore their messages sent in the channel. To setup the suggestion system, you may need Administrator permissions.

Command Example Usage
suggestionchannel add <channel> /suggestionchannel remove channel:suggestion Sets a channel to be a suggestion channel of the server.
suggestionchannel remove <channel> /suggestionchannel remove channel:suggestion Removes a channel to be a suggestion channel of the server.

The following slash commands are for setting up sticky messages in different channels. Sticky messages will be stayed at the bottom of the channel no matter what messages are sent afterwards. This function is especially useful when you need give your members a heads up on how to use this chat channel or what they need to know before going ahead and sending any messages. To setup sticky messages, you may need Administrator permissions.

Command Example Usage
sticky enable <message> /sticky enable message:this channel is used for emergency purposes only Enables the sticky message in the channel.
sticky disable /sticky disable Disables the sticky message in the channel.
